Scale Factor

To limit the scale factor, just set the CSS properties [min/max]-width and [min/max]-height to the .red-zoom__lens-image element.

.red-zoom__lens-image {{'{'}} --red-zoom-min-scale-factor: 1; --red-zoom-max-scale-factor: 2; min-width: calc(var(--red-zoom-thumb-w) * var(--red-zoom-min-scale-factor)); min-height: calc(var(--red-zoom-thumb-h) * var(--red-zoom-min-scale-factor)); max-width: calc(var(--red-zoom-thumb-w) * var(--red-zoom-max-scale-factor)); max-height: calc(var(--red-zoom-thumb-h) * var(--red-zoom-max-scale-factor)); {{'}'}}

For these purposes, you can also use the prepared SASS mixins: red-zoom-[min/max]-scale-factor. The first argument is the value of the scale factor, the second is the element relative to which the scale factor is specified, the possible values are 'thumb' (default) and 'lens-image'.

{{"@import '~ngx-red-zoom/styles/mixins/common.scss';"}} // The enlarged image cannot be smaller than the thumbnail. {{"@include red-zoom-min-scale-factor(1, 'thumb');"}} // The enlarged image cannot be larger than its natural size. {{"@include red-zoom-max-scale-factor(1, 'lens-image');"}}
