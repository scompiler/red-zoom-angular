Size And Position Of The Lens

You can set any size of the lens image via CSS/SCSS, red-zoom automatically detect it, and you do not need to edit ts/html files for this purpose. This also means that you can set different sizes of the lens depend on screen size using CSS media queries.

Use the following CSS variables to position the lens relative to the thumbnail:

--red-zoom-thumb-[x/y] Thumbnail position relative to the page in pixels.
--red-zoom-thumb-[w/h] Thumbnail size in pixels.
--red-zoom-thumb-size-[max/min] Max/min size of thumbnail in pixels.
