Customization And Styles

RedZoom comes with three predefined styles, you can use one of them as shown below:

// sass @use 'ngx-red-zoom/styles/base.scss'; @use 'ngx-red-zoom/styles/style-{{ style.value }}.scss'; <!-- html --> <img src="./assets/image.jpg" redZoom="./assets/image-full.jpg" redZoomClass="red-zoom--style--{{ style.value }}" >

You can also create your own amazing style or customize an existing style using mixins:

// sass @use 'ngx-red-zoom/styles/base.scss'; @use 'ngx-red-zoom/styles/mixins/style-{{ style.value }}.scss'; .red-zoom--style--custom {{ '{' }} @include style-{{ style.value }}.red-zoom-style-{{ style.value }}( // you settings here ); }
